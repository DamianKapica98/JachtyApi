W celu uruchomienia strony lokalnie trzeba:
stworzyć bazę danych z załączonego skryptu
Uruchomić projekt Api/Jachty_TI/Jachty_TI.sln
Zmapować bazę w Web.config <connectionStrings> (zmienić Data Source)
Uruchomić rozwiązanie
Przejść do folderu Api/UI/AngularUI i uruchomić w nim CMD
(opcjonalne) w celu zobaczenia kodu html i typescript wpisać w konsoli ". code"  (wymagany visual studio code)
Jeżeli mamy zainstalowanego angulara do uruchomienia strony wystarczy polecenie "ng serve --open"
Jeżeli wszystko jest poprawnie po kliknięciu przycisku "Jachty" załadują się zdjęcia oraz jachty z bazy danych
Zakładka rezerwacja powinna umożliwia rezerwację oraz podgląd już dokonanych rezerwacji
